### Usage
#### To crawl data
```commandline
python3 SearchingSynonym.py --crawl-data
```
#### To get synonyms
```commandline
python3 SearchingSynonym.py --get-data happy
```
Thay "happy" bằng từ khác để lấy dữ liệu.
### Option
```
--host "localhost"
--port "3306"
--user "root"
--pass-word ""
--database "" --> Thay bằng tên database
```
